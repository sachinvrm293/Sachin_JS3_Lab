AJAX Introduction

Asynchronous JavaScript And XML

Ajax is not a programming language

Ajax just uses a combination of

  • Browser built in XMLHttpRequest Object
  • To request the data from a Web Server!
  • JavaScript and HTML DOM
  • to display or use the data

How AJAX Communication happens?

  1. An event occurs in a web page (the page is loaded, a button is clicked)
  2. An XMLHttpRequest object is created by JavaScript
  3. The XMLHttpRequest object sends a request to a web server
  4. The server processes the request
  5. The server sends a response back to the web page
  6. The response is read by JavaScript
  7. Proper action (like page update) is performed by JavaScript

HTTP Methods

    1. GET :- Get the results from backend
    2. URL:- http://localhost:8080/users
    3. DATABASE Query:- select * from user;
    4. HTTP Response :- [{id:1, "raghav"}, {id:2, "keshav"}, {id:3, "madhav"}]
    5. HTTP Status Code :- Sucess :- 200
    6. HTTP Status Code :- Failure :- 500 / 400 etc..
    1. POST :- Create a record in the backend
    2. URL:- http://localhost:8080/users + RequestBody[{id:1, "raghav"}]
    3. DATABASE Query:- insert into table user(id, name) values(3, "thilak");
    4. HTTP Response :- [{id:1, "raghav"}]
    5. HTTP Status Code :- Sucess :- 201
    6. HTTP Status Code :- Failure :- 500 / 400 etc..
    1. PUT :- Modify a record in the backend
    2. URL:- http://localhost:8080/users/1 + RequestBody[{id:1, "raghavendra"}]
    3. DATABASE Query:- update user set name="raghavendra" where id=1
    4. HTTP Response :- [{id:1, "raghavendra"}]
    5. HTTP Status Code :- Sucess :- 200
    6. HTTP Status Code :- Failure :- 500 / 400 etc..
    1. DELETE :- delete a record in the backend
    2. URL:- http://localhost:8080/users/5
    3. DATABASE Query:- delete user where id=5
    4. HTTP Response :- [{id:5, "shanthanu"}]
    5. HTTP Status Code :- Sucess :- 200
    6. HTTP Status Code :- Failure :- 500 / 400 etc..
